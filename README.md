# CUDA Image Processing at Scale

## Description
This project processes a large batch of PPM images using CUDA to convert them to grayscale.
It utilizes GPU parallelism to process pixels efficiently.

## Structure
- src/: Source code
- input_data/: Input PPM images
- output_data/: Processed images

## Compilation
`make`

## Execution
`./image_proc -i input_data -o output_data`
